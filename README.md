# 🛡️ Fake Virus Prank Script

סקריפט Batch פשוט למטרות **שעשוע ומתיחה בלבד**. הסקריפט מדמה תהליך של "פריצה" והרס של המחשב בעזרת אפקטים ויזואליים, מבלי לבצע שום שינוי או נזק לקבצים שלכם.

---

## 📋 מה הסקריפט עושה?

1. **הסוואה ראשונית:** מציג מצג שווא של התקנת Google Chrome.
2. **הודעת "ווירוס":** משנה את צבע המסך ומציג הודעות שגיאה ומחיקה של תיקיית `System32`.
3. **הפעלה מחדש של הממשק:** מבצע `taskkill` ל-`explorer.exe` ומחזיר אותו כדי לגרום למסך "להבהב" וליצור תחושת תקלה.
4. **מצב בהלה (PowerShell):** מריץ קוד שגורם לעכבר לזוז לבד וממלא את המסך בכיתוב אדום של המילה **ERROR**.
5. **סימולציית מחיקה:** מריץ את פקודת ה-`tree` שמציגה רשימת קבצים רצה במהירות כדי לדמות מחיקה המונית של הדיסק הקשיח.

---

## 🛠️ הוראות הפעלה

1. העתיקו את קוד ה-Batch לקובץ טקסט חדש (Notepad).
2. שמרו את הקובץ בשם לבחירתכם עם סיומת `.bat` (לדוגמה: `setup_chrome.bat`).
3. הריצו את הקובץ בלחיצה כפולה.

> **עצירה:** ניתן לסגור את חלון ה-CMD בכל רגע כדי להפסיק את פעולת הסקריפט.
